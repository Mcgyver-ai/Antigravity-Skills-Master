# Approved Documentation Sources

Next.js Official Docs: <https://nextjs.org/docs>
(Specifically look at the "App Router" section, as that is the modern standard).

React Official Docs: <https://react.dev/reference/react>
(For anything involving state, hooks, or components).

Node.js API: <https://nodejs.org/en/docs/>
(For your backend scripts and file reading).

shadcn/ui: <https://ui.shadcn.com/docs>
(For UI components and styling).

Tailwind CSS: <https://tailwindcss.com/docs>
(For styling rules and responsive design).

Framer Motion: <https://www.framer.com/motion/>
(For animations and micro-interactions).

Lucide React: <https://lucide.dev/icons/>
(For icons).

Zustand: <https://github.com/pmndrs/zustand>
(For state management).

Zod: <https://zod.dev/>
(For schema validation).

React Hook Form: <https://react-hook-form.com/>
(For form handling).

React Query (TanStack Query): <https://tanstack.com/query/latest>
(For data fetching and state management).

Radix UI: <https://www.radix-ui.com/>
(For unstyled, accessible components that shadcn/ui is built on top of).

React Server Components (RSC): <https://react.dev/reference/rsc>
(For understanding the App Router architecture).

TypeScript: <https://www.typescriptlang.org/docs/>
(For type definitions and safety).

Gemini API Docs (Google AI Studio): <https://ai.google.dev/docs>
(Look here for the "Vision" API docs when building your invoice extraction skill).

Google Cloud Console (for API Keys): <https://console.cloud.google.com/>

For Emails (e.g., Resend): <https://resend.com/docs>
(Highly recommended for Next.js apps to send those overdue reminders).

For NPM Packages: <https://www.npmjs.com/>
(Always check a package here first to ensure it has high weekly downloads and was updated recently before letting your agent install it).
