# Social Media Content Repurposer

## When to Use This Skill
Activate when the user:
- Asks to "turn this into a Twitter thread"
- Wants to adapt content for different platforms
- Needs to create multiple social posts from one source
- Says "repurpose this blog post"
- Wants platform-specific content variations

## Instructions
1.  **Analyze Source Content**: Identify main message, key points (3-5), tone, and quotable moments.
2.  **Optimize for Twitter/X**: Break into thread (1 hook + 5-10 tweets), <280 chars/tweet, strategic line breaks.
3.  **Optimize for LinkedIn**: Professional tone, 1300-1700 chars, strong hook, list format, discussion prompt.
4.  **Optimize for Instagram**: Concise message (125-150 words), visual hook first, 10-20 hashtags, visual suggestion.
5.  **Optimize for Facebook**: Conversational, 40-80 chars (or 200+), questions for engagement.
6.  **Optimize for TikTok**: Short script (30-60s), visual hook, trending audio, on-screen text.

## Best Practices
-   **Maintain Core Message**: Don't distort the original meaning.
-   **Adapt Tone**: Twitter is punchy, LinkedIn is professional, IG is visual.
-   **Strong Hooks**: Essential for every version to stop the scroll.
-   **Mobile Optimization**: Use line breaks and readable formatting.
-   **Native Features**: Use polls, threads, and platform-specific tools.
-   **Call to Action**: Always tell the audience what to do next.
