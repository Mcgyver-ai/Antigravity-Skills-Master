# Social Post Creator

## When to Use This Skill
- Sharing company updates on LinkedIn
- Engaging with community on Twitter/X
- Building brand awareness on Instagram
- Driving discussion on Threads
- Cross-platform campaigns

## Quick Reference
| Action | Command/Trigger |
| :--- | :--- |
| LinkedIn post | "Write LinkedIn post about [topic]" |
| Twitter/X thread | "Create Twitter thread on [topic]" |
| Instagram caption | "Draft Instagram caption for [content]" |
| Threads post | "Write Threads post about [topic]" |
| Cross-platform | "Create social posts for all platforms about [topic]" |

## Core Workflows

### Workflow 1: Single Platform Post
1.  **Define Goal**: Understand objective (engagement, clicks, awareness).
2.  **Choose Platform**: Select primary social channel.
3.  **Craft Message**: Write platform-optimized content (hooks, structure).
4.  **Add Media**: Suggest images, videos, or links.
5.  **Optimize Hashtags**: Add relevant tags for discovery.
6.  **Schedule**: Determine optimal posting time.

### Workflow 2: Cross-Platform Campaign
1.  **Develop Core Message**: Create main theme.
2.  **Adapt per Platform**: Customize for each channel (don't copy-paste).
3.  **Maintain Consistency**: Keep key points across all versions.
4.  **Vary Format**: Use each platform's strengths (Visuals for IG, Text for LI).
5.  **Coordinate Timing**: Stagger or synchronize posts.
6.  **Track Performance**: Monitor which versions resonate.

### Workflow 3: Content Calendar Planning
1.  **Identify Themes**: Plan content pillars.
2.  **Map to Calendar**: Schedule posts in advance.
3.  **Balance Content Types**: Mix promotional, educational, entertaining.
4.  **Create Templates**: Build reusable formats.
5.  **Batch Create**: Write multiple posts efficiently.

## Platform-Specific Guidelines

### LinkedIn
*   **Limit**: 3,000 chars (optimal 150-300)
*   **Best Practices**: Lead with value, personal stories, professional tone.
*   **Format**: Hook -> Main Content -> Takeaways -> CTA -> Hashtags.

### Twitter/X
*   **Limit**: 280 chars (or longer with premium)
*   **Best Practices**: Front-load hook, one idea per post, use threads.
*   **Format**: Hook -> Context -> Points -> Conclusion/CTA.

### Instagram
*   **Limit**: 2,200 chars
*   **Best Practices**: Front-load value, use visual storytelling, 5-10 hashtags.
*   **Format**: Hook -> Story/Value -> Key Points -> CTA -> Hashtags.

### Threads
*   **Limit**: 500 chars
*   **Best Practices**: Conversational, engage quickly, mix text/images.
*   **Format**: Hook -> Point -> Question.

## Content Types & Templates
1.  **Announcement Post**: "Big news: [Announcement] 🎉... What this means:..."
2.  **Thought Leadership**: "Unpopular opinion: [Take]... Here's why:..."
3.  **Value/Educational**: "[Number] lessons from [experience]:..."
4.  **Engagement/Question**: "Quick question for [audience]:..."
5.  **Story/Case Study**: "[Hook about problem]... [Challenge]... [Solution]... [Results]"
6.  **Behind-the-Scenes**: "Real talk: [Honest admission] 😅..."

## Common Pitfalls
-   Over-Promoting (Aim for 80% value, 20% promo)
-   Being Too Generic
-   Ignoring Comments
-   Inconsistent Posting
-   Wrong Platform Tone
-   Hashtag Spam (Quality > Quantity)
